What this page is. Every finished Canva ad Paulina built, grouped by size. The sidebar on the left shows the element vocabulary — the real generator will drive these 6 fields. The images here are frozen final output, for reverse-engineering only.

How to annotate. Click any ad → pick an element type in the annotation bar below → drag on the image to draw its bounding box. Right-click a box to delete. Drag the corner handle to resize. Export spec.json when done.

Layout variants. Some sizes have multiple layout variants (full text / no subtitle / logo only / template). The element presence matrix — which fields appear in which variant — is what we're capturing.
